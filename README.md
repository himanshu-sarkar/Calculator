# Simple Python Calculator

This is my first Python CLI project built while learning Python and Git.

## Features

- Addition, Subtraction, Multiplication, Division
- Power (`**`), Percentage (`%`), Square Root (`sqrt`)
- Natural Logarithm (`log`) and Factorial (`fact`)
- Division by zero and invalid input handling
- Calculation history with timestamps
- History saves to a JSON file between sessions
- Runs continuously until you quit
